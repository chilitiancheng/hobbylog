# 个人兴趣生活主页

这是一个纯静态网站，可以直接部署到 GitHub Pages。

## 本地打开

- 直接双击 `index.html`
- 或双击 `start-local.bat`，用本地服务打开 `http://127.0.0.1:8790/`

## GitHub Pages 部署

把本目录里的这些文件放到 GitHub 仓库根目录：

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

然后在仓库 `Settings -> Pages` 中选择从默认分支部署即可。

数据默认保存在当前浏览器本地。换设备前，请在网站的“设置与备份”里导出 JSON。

推荐短仓库名：`xqj`。部署后的链接通常是：

`https://你的GitHub用户名.github.io/xqj/`
